============================================================
  双机实时音频监测系统 - 前端面板
============================================================

【文件说明】
  frontend-panel.html    前端采集面板（录音 + 文件上传 + 结果）
  backend-dashboard.html  后端实时监控仪表盘
  emotion-panel.html      情绪分析面板
  start.bat              一键启动（Windows）

============================================================
  使用方法
============================================================

1. 双击 start.bat
   → 自动启动本地 HTTP 服务器（端口 8081）
   → 自动打开浏览器

2. 点击右上角齿轮图标，配置后端地址
   → 例如: http://192.168.1.100:8080
   → 点击"测试连接"，显示"后端就绪"即可

3. 三个页面：
   frontend-panel.html    → 上传音频 / 录音分析
   backend-dashboard.html → 实时分类 + 报警监控
   emotion-panel.html     → 情绪分析详情

============================================================
  手动启动（备用）
============================================================

  python -m http.server 8081 --bind 127.0.0.1

  然后浏览器访问：
  http://localhost:8081/frontend-panel.html
  http://localhost:8081/backend-dashboard.html
  http://localhost:8081/emotion-panel.html

============================================================
  系统要求
============================================================

  前端电脑:
    - Python 3.10+
    - Chrome / Edge / Firefox
    - 麦克风（录音功能需要）

  后端电脑:
    - 需要先启动后端
    - 防火墙开放端口 8080
    - 前端电脑能 Ping 通后端电脑

============================================================
  提示
============================================================

  - 后端地址保存在浏览器本地，配置一次即可
  - 也可通过 URL 参数传入：
    ?backend=192.168.1.100:8080
  - 全程局域网通信，不需要互联网
  - 仅录音功能需要麦克风权限
